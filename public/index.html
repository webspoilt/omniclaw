üì± Ultimate Android / Termux WiFi Host Guide

Why leave a PC running when an old Android phone uses 2 Watts of power? You can turn any Android device into a 24/7 AI Node.


1. Install the Proper Termux

Do NOT download Termux from the Google Play Store (it is outdated and broken). You must download F-Droid, and install "Termux" from there.


2. Initial Setup & File Permissions

Open Termux and type:
termux-setup-storage
Accept the popup. Then update everything:
pkg update && pkg upgrade -y


3. Enable Remote Access (SSH) so you can type from your PC

Typing on a phone screen is hard. Let's connect it to your PC's keyboard via WiFi.

  • Install SSH server: pkg install openssh -y
  • Set a password: passwd (type a password, you won't see the letters)
  • Start the server: sshd
  • Find your phone's IP address: ifconfig (Look for the `wlan0` section, usually looks like 192.168.x.x)
  • Find your username: whoami (usually looks like u0_a123)
  • On your Windows/Mac PC open a terminal and type: ssh -p 8022 your_username@192.168.x.x

4. Install Dependencies & Ollama

Inside Termux install tools:
pkg install git python rust binutils nodejs-lts ollama -y
Start the AI engine:
ollama serve &


5. Install OmniClaw

Run the installer script:
curl -fsSL https://omniclaw.ai/setup.sh | bash

üõ†Ô∏è How the Linux Native Tools Work

OmniClaw is not a restricted web chatbot. It drops an autonomous Python `Worker` process directly into a terminal shell.

  • Tool Calling: The LLM outputs structured JSON asking to run commands like nmap -sV target.com or curl.
  • Execution: The OmniClaw orchestrator runs this in a sandboxed subprocess and feeds the stdout/stderr back into the LLM context limits.
  • Scripting: If a generic tool doesn't exist, OmniClaw is instructed to write a Python or Bash script to /tmp/, execute it, read the result, and delete it.

üß† The Peer-Reviewed Swarm Architecture

A single AI hallucinated often. OmniClaw chains APIs together to fix this.


The Hierarchy

  • The Manager (High IQ): You configure a powerful, expensive model (like GPT-4 or Claude-3.5) as the Manager. It reads your Telegram prompt and breaks it into 5 technical steps.
  • The Workers (Fast/Cheap/Local): The Manager assigns step 1 to a local free model (like Ollama Llama3 8B). The worker executes terminal commands to complete step 1.
  • The Reviewer: Before the final answer is sent to you, another model looks at the output and the terminal logs, and rates it. If it fails, the loop restarts.

üéôÔ∏è Desktop Voice Wake

Activating OmniClaw hands-free from across the room.

The core/voice_wake.py file uses Python's SpeechRecognition library. It runs continuously in the background using minimal CPU. When it hears the exact acoustic model for "Hey Omni", it triggers a generic software interrupt waking the main LLM loop. For Mac users, it hooks into the native macOS Accessibility dictation APIs.

Setup: Ensure your PC microphone is unmuted. In Linux/Raspberry Pi, ensure pyaudio and ALSA drivers are configured.

ü¶† eBPF Kernel Bridge (Advanced Linux Only)

Standard agents run in "user space" and are blind to the operating system. OmniClaw compiles a Rust/C eBPF binary that attaches directly to the Linux Kernel.

  • Network Visibility: The agent can intercept incoming DDoS packets or malicious SSH attempts at the socket layer before they even hit an application.
  • Process Tracing: It knows exactly when a file is opened or executed by any other program on the server.
  • Requirement: You MUST run OmniClaw as root on a Linux machine with kernel 5.8+ to use this module. It will not work on Windows without true WSL2 compilation.
