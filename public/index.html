üö® Experimental & Beta Software Warning

DO NOT run OmniClaw as root on your daily-driver host machine!

Because OmniClaw utilizes auto-compiling eBPF bridges and completely autonomous terminal execution loops, it has the power to modify kernel components.

If an underlying AI model hallucinates a destructive command (e.g., rm -rf), data loss will occur. You MUST run this software inside a heavily isolated Virtual Machine (VirtualBox, VMware, or Proxmox) or a strict Docker boundary. Treat the agent as an extremely powerful hazard.
